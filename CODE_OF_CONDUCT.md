# Code of Conduct

## Document Description

This document defines the morale code of the project *Open Quantum Computing Textbook* that any project maintainers and contributers must follow.

## Rules

1. The main language of communication is English. 
2. Any kind of hate speech is prohibited.

## The Use of AI

**Please refrain from using large language models (LLMs) to generate book contents from scratch.**
A contributer may utilize LLMs to check grammatical errors and polish their writings; however, the original text must be solely written by a human.
Meanwhile, the current AI dectection tools are not reliable to an extensive degree, any contribution attempts and/or pull requests with apparent use of LLMs will be rejected.
